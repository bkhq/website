# FE-003 删除 category 数据与路由并统一使用 tags 分类

- **status**: completed
- **priority**: P1
- **createdAt**: 2026-03-09 20:23
- **owner**: codex
- **completedAt**: 2026-03-09
- **relatedPlan**: PLAN-005

## 描述

彻底移除站点中的 `category` 概念，只保留 `tags` 作为内容分类方式：

- 删除 `content/list.json` 中的 `categories` 字典
- 删除每个工具列表项上的 `category` 字段
- 将原分类语义并入 `tags`
- 删除前端 `/category/*` 页面与相关筛选逻辑
- 详情页、列表页、关联项卡片不再依赖 category badge

## 验收标准

- [x] `content/list.json` 不再包含 `categories` 与 `list[].category`
- [x] 首页筛选与列表展示仅基于 tags
- [x] 详情页不再渲染 category badge
- [x] `/category/*` 页面与相关生成逻辑被删除
- [x] 构建通过
