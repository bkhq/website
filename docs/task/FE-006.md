# FE-006 拆分 tags 到独立 content/tags.json

- **status**: completed
- **priority**: P1

- **owner**: codex
- **completedAt**: 2026-03-09

## 描述

将站点级 tag 字典从 `content/list.json` 中拆分到独立的 `content/tags.json`：

- `content/list.json` 只保留 `featured` 与 `list`
- `content/tags.json` 维护 tag key 与中英文标签名
- 前端继续通过统一数据接口读取 tags，不改变页面使用方式

## 验收标准

- [x] `content/tags.json` 创建完成

- [x] 前端仍可正常读取 tag 列表并生成标签页
- [x] 构建和 lint 通过
