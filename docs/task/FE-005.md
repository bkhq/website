# FE-005 为工具内容目录增加 path 映射并支持嵌套目录

- **status**: completed
- **priority**: P1
- **createdAt**: 2026-03-09 20:31
- **owner**: codex
- **completedAt**: 2026-03-09
- **relatedPlan**: PLAN-007

## 描述

在不改变 URL slug 的前提下，为工具内容目录增加 `path` 映射字段，使内容可以放在嵌套目录中：

- `slug` 继续用于 URL 和站内引用
- `path` 用于指向 `content/` 下的实际目录
- 未配置 `path` 时默认回退到 `slug`
- 将 `zzci-chrome` / `zzci-traefik` 迁移到 `content/zzci/chrome` 与 `content/zzci/traefik`

## 验收标准

- [x] `list.json` 支持 `path`
- [x] 内容读取逻辑优先按 `path` 读取目录
- [x] `zzci-*` 内容迁移到嵌套目录后仍可正常构建
- [x] URL slug 保持不变
