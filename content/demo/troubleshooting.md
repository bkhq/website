If an export fails, check write permissions for your destination folder and confirm the image format is supported. For reproducible issues, visit the GitHub repository and include your operating system, app version, and a sample file when filing a report.

如果导出失败，请检查目标文件夹的写入权限并确认图片格式是否受支持。对于可复现的问题，请访问 GitHub 仓库，在提交报告时附上操作系统、应用版本和示例文件。
